ttylinux downloads - BeagleBone

The BeagleBone is a low-cost, high-expansion arm-based BeagleBoard: link

A wiki page covering the ARM architecture is here.

The current ttylinux release is 2014.10 for the (TI Sitara) ARMv7 architecture BeagleBone platform. The SDCard tarball is available here. The updated ChangeLog file is available:
⇒ ChangeLog-beagle_bone

This is a large system running from an SDCard; it has Lua scripting and a GCC-based C, C99, C++ development capability.
