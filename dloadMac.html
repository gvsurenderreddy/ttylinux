ttylinux downloads - Macintosh G4

The current ttylinux release is 16.1 for the PowerPC architecture Macintosh platform. The CD-ROM ISO image is available here. The updated ChangeLog file is available:
⇒ ChangeLog-mac_g4

This PowerPC architecture for ttylinux is a generic 32-bit PowerPC G3 or G4 (74xx/75xx), intended to be suitable for Apple PowerMac and PowerBook computers.

This is a large system running in a 96 MB RAM disk; it has VESA, Intel, ATI and Nvidia frame buffers, GPM for mouse, ALSA sound, Lua scripting, GCC-based C development capability, a small web server, and the haserl cgi scripting program.

Binary Distribution and Source ISO
The CD-ROM ISO image file is the ttylinux release format; it is directly bootable after burning it onto a suitable blank disc. The ISO image also has all the available ttylinux documentation, kernel symbols and symbol map file, and the individual ttylinux binary packages.
NOTE The source ISO has only the source packages available at the above links; it has no build or configuration scripts and is not used by ttylinux
