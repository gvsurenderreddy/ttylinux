ttylinux downloads - PC x86_64

The current ttylinux release is 16.1 for the PC x86_64 platform. The CD-ROM ISO image is available here. The updated ChangeLog file is available:
⇒ ChangeLog-pc_x86_64

The User Guide applies to PC architecture versions of ttylinux.

This is a large system running in a 96 MB RAM disk; it has VESA, Intel, ATI and Nvidia frame buffers, GPM for mouse, ALSA sound, Lua scripting, GCC-based C development capability, a small web server, and the haserl cgi scripting program.

Binary Distribution and Source ISO

The CD-ROM ISO image file is the ttylinux release format; it is directly bootable after burning it onto a suitable blank disc. The kernel has support for most PCI IDE, SATA (Serial ATA) controllers and most available PCI network cards. The ISO image has all the available ttylinux documentation, kernel symbols and symbol map file, and the individual ttylinux binary packages.

NOTE The source ISO only has the same source packages available at the above links. It is NOT used by ttylinux, has no build or configuration scripts, and you very probably do not need it. It is here for GPL compliance.
