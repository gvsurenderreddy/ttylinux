ttylinux downloads - Virtio x86_64

The current ttylinux release is 16.1 for the Virtio x86_64 platform. The CD-ROM ISO image is available here. The updated ChangeLog file is available:
⇒ ChangeLog-virtio_x86_64

This ISO boots into a 64 MB RAM disk. Unlike the PC x86_64 ttylinux, this virtio system has no frame buffers, no GPM for mouse and no ALSA sound. It does have Lua scripting, GCC-based C development capability, a small web server, and the Haserl CGI scripting program.

The expected use for this ISO is to boot it in a Virtual Machine and install onto a virtual disk. The ttylinux-installer program should be used to install the virtio ttylinux onto a virtual disk. The root directory has an installation config file; it should be carefully and completely read then edited for your use, and then used to install from the CDROM onto the virtual disk:
$ ttylinux-installer --config=install.conf /dev/sr0

Binary Distribution and Source ISO

The CD-ROM ISO image file is the ttylinux release format; it is directly bootable after burning it onto a suitable blank disc. The ISO image has all the available ttylinux documentation, kernel symbols and symbol map file, and the individual ttylinux binary packages.

NOTE The source ISO only has the same source packages available at the above links. It is NOT used by ttylinux, has no build or configuration scripts, and you very probably do not need it. It is here for GPL compliance.
