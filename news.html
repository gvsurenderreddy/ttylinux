2013-05-11

Virtio ttylinux releases final 16.0 are available for:
⇒ Virtio i486
⇒ Virtio i686
⇒ Virtio x86_64

These new releases are bootable ISO images; boot and install onto a virtual hard disk in a VM.



2013-04-25

New virtio ttylinux releases 16.0 are available for:
⇒ Virtio i686
⇒ Virtio x86_64

These new releases are bootable ISO images; boot and install onto a virtual hard disk in a VM. If you use one of these in a virtual environment and you find something missing or wrong, please let me know.

These new ttylinux systems have Gcc-based C development tools, Lua scripting, a small http server, and Haserl CGI scripting.



2013-04-08

⇒ PC i486
⇒ PC i686
⇒ PC x86_64
⇒ Macintosh G4 (PowerPC 74x/75x)
⇒ BeagleBone ARM Cortex-A8 (TI Sitara AM335x SoC)
⇒ WRTU54G-TM (MIPS)

Added Haserl CGI scripting to the Macintosh G4, PC i686, and PC x86_64 platforms. The master branch in the github repository is up-to-date; no need to check any other branches.

I am now working on virtio platforms for i686 and x86_64.
