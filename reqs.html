ttylinux
requirements and features

The User Guide has a more complete description of the system requirements and a full list of the packages comprising ttylinux. The software features are generally the same for all CPU architectures. See the individual download pages for each system's hardware requirements and specific features.

Hardware Requirements - i486
  • 486SX processor or better
  • 28 MB RAM for general use, and 32 MB RAM for performing installation
  • 12 MB partition for hard drive installation
  • Ethernet interface, for high-speed Internet access
Hardware Requirements - i686, x86_64
  • PentiumPro processor or better for ttylinux-i686
  • 192 MB RAM for general use and performing installation
  • 96 MB partition for hard drive installation
  • Ethernet interface, for high-speed Internet access
General Software Features
  • command line interface, no graphical user interface
  • ready for Ethernet Internet access
  • firewalled Internet
  • text mode web browser
  • secure shell client and server
  • FTP and TFTP clients and servers
  • vi text editor
  • gcc and make with ttylinux-i686 and ttylinux-x86_64
  • package manager for easy installation of additional software
  • installer for CD-ROM based installation to hard disk
  • all packages updated on a regular basis
